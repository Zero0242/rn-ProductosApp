# Productos App

Aplicacion conectada con server, para pruebas con react native

## Dependencies

1. AsyncStorage

```bash
$ @react-native-async-storage/async-storage
```

2. React Picker

```bash
$ yarn add @react-native-picker/picker
```

3. React Image Picker

```bash
$ yarn add react-native-image-picker
```
